# Financial-Modelling-Valuation-Infosys-Excel

Financial Modelling & DCF Valuation of Infosys Ltd. using Microsoft Excel, featuring 10 years of historical financial analysis, ratio analysis, financial forecasting, and intrinsic valuation.

This project presents a comprehensive financial model and valuation of Infosys Ltd. built in Microsoft Excel. The analysis covers 10 years of historical financial statements, common-size analysis, ratio analysis, financial forecasting, and intrinsic valuation using the Discounted Cash Flow (DCF) method. The model follows a structured approach commonly used in equity research and corporate finance to evaluate a company's financial performance and estimate its intrinsic value.
